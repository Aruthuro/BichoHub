# EP06 - Sistema de Plantões e Disponibilidade

[Voltar para o Hub de Épicos](./EP00%20-%20Epicos%20do%20projeto.md)

## Objetivo

Gerenciar disponibilidade operacional dos coletores.

## E40 - Cadastro de horários de plantão

- [ ] Definir horário disponível por semana;
- [ ] Definir horário disponível por calendário;
- [ ] Escolher anonimização de informação na visualização pública da escala.

## E41 - Visualização pública de disponibilidade

- [ ] Disponibilizar horários cobertos por um coletor.

## E42 - Atualização em tempo real dos plantões

- [ ] A cada acesso ao servidor, o frontend deve pedir atualização de escala. 

## E43 - Associação de coletores por área/região

- [ ] A cada área demarcada, atribuir coletores autorizados a atuar.
- [ ] Um coletor só pode acessar o horário de plantão de áreas autorizadas;

## E44 - Controle de permissões para edição

- [ ] Administradores de área podem editar horários de outros coletores.
